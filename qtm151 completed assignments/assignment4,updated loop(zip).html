To answer the 1st question:

The number of simulations is correlated with the graph's "discreteness." If I run the graph with a very low number (such as 5) then the graph will look very discrete. It is not continuous, because there is a visible gap in between.

However, if I increase the number of simulations, the graph seems continuous. (discrete distribution versus continuous distribution)

On the other hand, if we fix the number of simulations while changing the number of sample sizes, the range decreases. The comparisons below show this difference: when sample size is merely 10, you have the range approximately 2, but when sample size is increased to 1000, the range is only about 0.2. Also, in probability, when you increase sample sizes, the distribution is becomes more similar to a normal distribution.
